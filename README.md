## About 4K Image Compressor

4K Image Compressor is a Windows desktop utility made to reduce and optimize images without turning the job into a repetitive manual process. It is geared toward large or high-resolution files and helps when you want to prepare images for the web, send them to others, or reclaim disk space.

Batch handling is part of the workflow, so you can drop in several files and process them together instead of repeating the same steps one at a time. Because compression runs locally and offline, everything stays on your own machine, keeping the experience simple and self-contained.

---

## Features

- Shrink high-resolution images to smaller file sizes
- Handle multiple files at once with drag-and-drop batch processing
- Fine-tune output with a custom quality slider
- Convert across common formats such as JPEG, PNG, and WebP
- Switch between lossy and lossless compression modes
- Keep or strip EXIF metadata during processing
- Preview images before committing the final output
- Run everything locally on Windows without requiring an internet connection

## Configuration

All settings are adjusted from within the application interface. Typical controls include compression level, output format, and EXIF behavior.

Example of the sort of workflow settings you can set:

    format: webp
    mode: lossy
    exif: keep

Any preference changes are applied directly in the desktop utility rather than through a separate command-line configuration file.
